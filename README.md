## ✨ Overview

**Flavor** is a sleek, fully responsive digital agency landing page template crafted entirely with **semantic HTML5** and **modern CSS3**. No JavaScript. No build tools. No frameworks. Just clean, hand-written code that demonstrates mastery of front-end fundamentals.

## 🚀 CSS Techniques Demonstrated

- **CSS Custom Properties** — Centralized design tokens for colors, spacing, and more
- **CSS Grid & Flexbox** — Responsive layouts without a single media query hack
- **Glassmorphism** — `backdrop-filter: blur()` on the navbar
- **Gradient Text** — `background-clip: text` for stunning gradient headings
- **CSS Animations** — `@keyframes` for floating shapes and fade-in effects
- **Pure CSS Hamburger** — Checkbox hack for a fully functional mobile menu (no JS!)
- **Pseudo-elements** — `::before` animated gradient borders on service cards
- **Fluid Typography** — `clamp()` for perfectly responsive font sizes
- **CSS Transitions** — Smooth hover states on every interactive element
- **Responsive Design** — 4 breakpoints covering desktop → small mobile

## ⚡ Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/YOUR_USERNAME/flavor.git
   ```

2. **Open in browser**
   ```bash
   cd flavor
   start index.html        # Windows
   open index.html          # macOS
   xdg-open index.html     # Linux
   ```

   Or use **Live Server** in VS Code for hot reload.

That's it. No `npm install`. No build step. No config files.

## 🎨 Customization

All design tokens are defined as CSS custom properties at the top of `style.css`:

```css
:root {
    --clr-bg: #0a0a0f;
    --clr-accent-1: #7c5cfc;     /* Primary purple */
    --clr-accent-2: #00d4ff;     /* Secondary cyan */
    --clr-gradient: linear-gradient(135deg, var(--clr-accent-1), var(--clr-accent-2));
    /* ... */
}
```

Change these values and the entire site updates instantly. Want a blue theme? A green theme? Just swap the accent colors.
